# lottery

## 介绍
浩哥的彩票生涯，用于预测大乐透和双色球。</br>
开始于2019年9月24日

## 规则
 **玩------法：** 五个红球两个篮球，红球取值1-35，篮球取值1-12。</br>
 **开奖时间--：** 每周一、三、六 21:10(网络直播 20:30)</br>
 **投注方式--：** 单式投注、复式投注、胆拖投注</br>
 **单注价格--：** 2元、追加3元</br>
 **单注最高奖：** 1800万(追加投注）</br>
 **奖------组：** 21425712个</br>
